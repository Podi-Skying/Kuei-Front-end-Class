CSS text語法

1. 文字顏色與排版

color: rgb(0 0 0 / 0.5); /* / 0.5 是指透明程度 */ /* 十六進位的顏色 紅色,綠色,藍色 */ /* color: #ff0000; color: #00ff00; color: #0000ff; */ /* 文字平均置中 left, center, right, justify*/ text-align: justify; /* 句尾 靠左或右 */ text-align-last: right; /* 文字距離 */ /* 字母距離: */ letter-spacing: normal; /* 單字距離: */ word-spacing: 10px; /* 行高: */ line-height: 50px; /* 首行縮排: */ text-indent: 64px; /* 所有網頁字體大小預設值為16px */ font-size: 16px;

2. 文字線條裝飾

text-decoration: red overline dotted 2px;

顏色: text-decoration-color: ;

樣式: text-decoration-style: ;

位置: text-decoration-line: ;

大小: text-decoration-thickness: ;

3. 字體及其大小

(1) em

font-size: 1em;

(2) rem

font-size: 1em;
rem = root的em
只有 :root 跟 html 可以動rem的值

font-weight: bold; /* normal, bold, bolder, light, lighter, 100-900 */ font-size: italic; /* normal, italic */ font-family: 'Nunito', 'Noto Sans TC'; /* serif 襯線字 */ /* sans-serif 非襯線字 */ /* 字體要加上引號，要先套英文 Nunito 再套中文 Noto Sans TC 字體 */
